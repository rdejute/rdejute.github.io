# Elevator Pitch — Script #1 (revised per feedback)

**Author:** Raina DeJute
**Module:** 16 — Professional Development
**Target length:** ~45–60 seconds (one elevator ride)

Hello, nice to meet you. My name is Raina DeJute.

Before I wrote code, I spent years in human services — guiding people through the hardest
transitions of their lives: recovery, crisis, starting over. I learned to really hear what people
need, not just what they say.

I build full-stack applications now, and that instinct shapes how I work. I'm comfortable across the
stack — React and React Native on the front, Spring Boot and Node on the back, SQL and MongoDB
underneath. But I'm still happiest in the moment a user finally feels understood.

Outside of coursework, I built a crypto analytics dashboard in React with a risk model that uses
logarithmic regression to turn noisy market data into a signal a person can actually read. I've also
built a tool that generates my grocery list and sequences it to match my real path through the
store — turning a scattered errand into one efficient loop. At the core of both is the same instinct:
meet a real human need with a tool that fits how people actually move.

For me, empathy isn't just a soft skill — it's my framework. It's how I catch the friction other
developers walk right past.

I'm looking for a full-stack team that treats thoughtful front-end work as the product, not the
afterthought.
